MySQL 基础

MySQL的体系结构

MySQL是客户/服务器体系结构。

  • 服务器程序：mysqld
  • 客户端程序：mysql，mysqldump，mysqladmin,mysqlimport,mysqlshow

启动/关闭数据库：

启动服务：
$ sudo /usr/local/mysql/support-files/mysql.server start

关闭服务:
$ sudo /usr/local/mysql/support-files/mysql.server stop

连接/断开接数据库

$ mysql -h hostname -u username -p

连接本地主机：
$ mysql -u username -p

断开连接：
exit

执行SQL语句

  • mysql>SELECT NOW();
  • mysql>SELECT NOW()\g
  • mysql>SELECT NOW()\G(竖排显示结果，每行一个值)
  • mysql> SELECT NOW(),USER(),VERSION();
  • mysql> SELECT NOW(),VERSION()\c(\c是取消已经输入的命令)
  • % mysql < myscript.sql(把命令写成一个脚本去执行)

注：命令（例如SELECT)和函数（例如USER(),NOW(),VERSION())不区分大小写。

创建数据库

  1. 创建（初始化）一个数据库。
  2. 在数据库里创建各种表。
  3. 对数据库表里的数据进行插入，检索，修改，删除等操作。
SQL语句：
mysql> CREATE DATABASE sampdb;

创建数据表

CREATE TABLE tbl_name (column_specs);

CREATE TABLE语句中的数据列定义由以下几个部分组成：数据列的名字，数据类型和一些属性。

一些常用的数据类型和属性

  • VARCHAR(n):可变字符（串），最多n个字符。
  • DATE:日期。标准形式：CCYY-MM-DD。
  • INT:整数。
  • UNSIGENED:不允许出现负数。
  • NULL:无数据。（可以为空值的意思）
  • NOT NULL:与NULL对应，不得为空值。
  • AUTO_INCREMENT:表示数据列里存放的是序列号，会为新插入输入自动加一付值。
  • PRIMARY KEY:主键，要求值必须唯一。隐含由NOT NULL属性。
